##  myvocab - это приложение командной строки.
Приложение в интерактивном режиме запрашивает у пользователя базовую директорию 
для анализа (парсинга) текстовых файлов. В результате работы приложения 
в выбранной директории создается каталог `Myvocab_58b254sv` со сформированными данными.

Перед запуском приложения необходимо выполнить действия обозначенные в файле `README.md` проекта:

- Installing tools
- Installing myvocab
- Building and executing myvocab.exe (опционально)
