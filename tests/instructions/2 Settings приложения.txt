`myvocab` - это приложение командной строки.
При запуске приложение запрашивает у пользователя базовый каталог 
для анализа (парсинга) текстовых файлов. В результате работы приложения 
в выбранной директории создается каталог `Myvocab_58b254sv` с сформированными данными.

Одним из сформированных файлов в созданной директории является файл настроек:

	базовый_каталог\Myvocab_58b254sv\settings.txt

Это единственный файл в базовом каталоге, доступный пользователю для редактирования и обмена данными с приложением.
При удалении каталога `базовый_каталог\Myvocab_58b254sv` все изменения в `settings.txt` будут утеряны.
Новый запуск приложения создаст этот каталог и заполнит `settings.txt` данными проекта.

Также одноименный каталог `Myvocab_58b254sv` создается в директории пользователя `Документы`.
Файлы в директории `Документы\Myvocab_58b254sv` используются как справочные данные при изменении формы слов.
Подробную информацию по справочным данным в директории `Документы\Myvocab_58b254sv`
смотреть в файле: `3 Справочные данные приложения.txt`


Первый запуск приложения заполняет файл настроек settings.txt данными по умолчанию:

--------------------------------------------------
Содержимое файла settings.txt:
--------------------------------------------------
result_file = vocabulary.txt
directories_file = directories.txt
use_lemma_singular = True
use_lemma_infinit = True
use_word_translate = False
use_order_text = True
use_folder_with_leading_exclamation_mark = False
------------------------------------------------

result_file - Наименование формируемого файла vocabulary.
directories_file - Наименование файла содержащего древовидную структуру парсинга каталогов начиная с базового.
use_lemma_singular - Флаг изменения множественной формы `отдельных английских слов`* на форму единственного числа.
use_lemma_infinit - Флаг изменения формы прошедшего времени `отдельных английских глаголов` на базовую форму.
use_word_translate - Флаг перевода `отдельных английских слов` на русский язык.
use_order_text - Флаг сортировки текста в формируемом файле vocabulary.
use_folder_with_leading_exclamation_mark - Флаг использования в парсинге папок и файлов, имя которых начинается с "!".


* Примечание: `отдельные английские слова` - это слова полученные из текста файла,
окруженного строковыми тегами <<word>> и <</word>> (отдельная строка с тегом).

Примечание:
При запуске приложения из командной строки с флагом -D:
    python myvocab.py -D
    myvocab.exe -D
отладочные логи, в том числе изменения слов, записываются в файл app.log каталога `Myvocab_58b254sv`.
В этом же режиме отладки -D, в папку `Translate` записываются все слова отправленные на перевод и полученный перевод.


В случае, если пользователь вносит некорректные данные, то они не сохраняются и выводится сообщение об этих данных. 
Допустимо при редактировании вместо True или False использовать соответственно 1 или 0.


Предварительно, перед запуском приложения, необходимо внести изменения в текстовые файлы парсинга.
Текст, который необходимо преобразовать в `отдельные английские слова`,
окружить строковыми тегами <<word>> и <</word>> (отдельная строка с тегом).
Текст, который не окружен этими тегами будет сохранен в `vocabulary` как он есть (кроме пустых строк).
Допустимо не указывать закрывающий тег <</word>> в конце файла.
Допустимо не поддерживать парность тегов <<word>> и <</word>>. Теги учитываются только при смене их значения.


Детализация работы флага `use_order_text`:

Если `use_order_text` равен True, то сортировка и уникальность применяется ко всему формируемому `vocabulary`.
Если `use_order_text` равен False, то сортировка и уникальность применяется к каждому файлу отдельно
в рамках формируемого `vocabulary`, с включением имен этих файлов.
