На основе общих правил образования множественного числа в английском, в программе
сформированы правила обратного преобразования `отдельных английских слов`* множественного числа в единственное.

* Примечание: `отдельные английские слова` - это слова полученные из текста файла,
окруженного строковыми тегами <<word>> и <</word>> (отдельная строка с тегом, см. пример ниже)


Используемые сокращения:

'ИД': Идентификатор правила (например, ИД=10) соответствует номеру операции в программном коде.
'Спр_А': Справочные данные из файла 'irregular_plural_nouns.txt' содержат существительные, 
чьи формы множественного числа нетипичны для соответствующих форм единственного числа.
'Спр_Б': Справочные данные из файла 'only_ending_s.txt' содержат существительные, 
которые оканчиваются на -s и не допускают изменение этого окончания, чтобы придерживаться 
правильной орфографии или общепринятого значения слова. 
'Спр_В': Справочные данные из файла 'singular_ending_non_s.txt' содержат существительные 
в начальной форме (в единственном числе) с заданным окончанием: -fe, -f -o, -y, -ch, -sh, -x, -z.

Примечание 1:
При запуске приложения из командной строки с флагом -D:
    python myvocab.py -D
    myvocab.exe -D
отладочные логи, в том числе 'ИД' преобразований, записываются в файл app.log каталога `Myvocab_58b254sv`.

Примечание 2:
Все справочники заполнены ограниченным количеством вариантов.
Поэтому по мере работы с программой может возникнуть необходимость в ручном дополнении
этих справочников в том же формате, в каком они уже частично заполнены.

Примечание 3:
Справочник 'Спр_В' достаточно заполнять существительными только с окончанием -fe
для правильной отработки правила `заменить -ves на -fe`.
Также этот справочник работает как "перехватчик" всех некорректных преобразований окончания -s.
Для этого достаточно занести в этот справочник базовую форму слова без окончания -s.
Другие случаи с окончаниями: -fe, -f -o, -y, -ch, -sh, -x, -z применяются опционально.

Примечание 4:
Справочник 'Спр_В' может содержать существительные единственного числа с окончаниями:
	-fe, -f;  -o; -o; -y; -y; -ch; -sh; -x; -z (и др. окончания после отбрасывания -s)
Программа на их основе преобразует существительные множественного числа с окончаниями:
	-ves (-fe, -f); -oes (-o); -os (-o); -ies (-y); -ys (-y); -ches (-ch); -shes (-sh); -xes (-x); -zes (-z).

Варианты использования всех справочников представлены ниже.	

Чтобы начать обработку отдельных английских слов, в текстовый файл необходимо добавить строку с тегом <<word>>:

<<word>>

1) Множественное число — исключения ('ИД'=10) ('Спр_А')

а) Некоторые существительные имеют нестандартную форму множественного числа:
    women -> woman (женщины -> женщина)
    men -> man (мужчины -> мужчина)
    children -> child (дети -> ребенок)
    feet -> foot (ступни -> стопа)
    mice -> mouse (мыши -> мышь)
    geese -> goose (гуси -> гусь)
    quizzes -> quiz (викторина -> викторины)


б) Слова латинского и греческого происхождения во множественном числе изменяются тоже нестандартно:
    schemata -> schema (схемы -> схема)
    dogmata -> dogma (догмы -> догма)
    phenomena -> phenomenon (явления -> явление)
    criteria -> criterion (критерии -> критерий)
    crises -> crisis (кризисы -> кризис)
    theses -> thesis (тезисы -> тезис)
    formulae -> formula (формулы -> формула)
    millennia -> millennium (тысячелетия -> тысячелетие)
    data -> datum (данные -> данное)
    octopi -> octopus (осьминоги -> осьминог)


в) Некоторые существительные латинского и греческого происхождения имеют
две формы множественного числа: оригинальную и английскую:
    formulae/formulas -> formula (формулы -> формула)
    schemata/schemas -> schema (схемы -> схема)
    curricula/curriculums -> curriculum (учебные планы -> учебный план)
    millennia/millenniums -> millennium (тысячелетия -> тысячелетие)
    referenda/referendums -> referendum (референдумы -> референдум)


2) Существительные, которые оканчиваются на -s и не допускают изменение этого окончания, 
чтобы придерживаться правильной орфографии или общепринятого значения слова ('ИД' = 20) ('Спр_Б'): 

а) грамматически множественное число:
    scissors (ножницы), glasses (очки), headphones (наушники), jeans (джинсы),
    earnings (заработок), clothes (одежда), goods (товар), stairs (лестница) ...


б) грамматически единственное число:

в) число меняется в зависимости от контекста:
    series (серия), species (вид), crossroads (перекрестки) ...
