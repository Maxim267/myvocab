`myvocab` - это приложение командной строки.
Приложение в интерактивном режиме запрашивает у пользователя базовую директорию 
для анализа (парсинга) текстовых файлов. В результате работы приложения 
в выбранном директории создается каталог `Myvocab_58b254sv`
со сформированными данными.

Перед запуском приложения необходимо выполнить действия обозначенные в файле README.md проекта:

- Installing tools
- Installing myvocab
- Building myvocab.exe (опционально) 


Интерактивный режим работы приложения на Python:

	При запуске `myvocab` выводится сообщение:
	
	> The root directory for parsing text files: C:\путь\к\папке
	> Do you want to continue? [Y/n]:

	Необходимо нажать `n`, если указанная папка не является целевой для парсинга текстовых файлов.
	Далее выводится сообщение:
	
	> Do you want to enter a new directory for parsing text files? [Y/n]:
	
	Если нажать `n` программа завершится.
	Если нажать `Enter` (по умолчанию Y) выводится сообщение:
	
	> New root directory for parsing text files:
	
	где необходимо ввести путь к базовой директории парсинга текстовых файлов, например,
	
	C:\путь\к\базовой\папке\парсинга
	
	Приложение в случае успешного завершения выведет:
	
	> INFO - The resulting vocabulary has been created:
	> путь\к\файлу\Myvocab_58b254sv\vocabulary.txt
	
	В случае критической ошибки, например, несуществующей директории, выведет сообщение:
	
	> CRITICAL -
    > ========================================
    > Uncaught exception: application will terminate.
    > Type: DirectoryNotExistError
    > Value: Directory does not exist: 'C:\неверный\путь\к\папке'
