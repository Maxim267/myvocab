На основе общих правил добавления окончания -ed к глаголам в базовой форме в английском, в программе
сформированы правила обратного преобразования `отдельных английских слов`* с окончанием -ed в базовую форму.

* Примечание: `отдельные английские слова` - это слова полученные из текста файла,
окруженного строковыми тегами <<word>> и <</word>> (отдельная строка с тегом, см. пример ниже)


Используемые сокращения:

'ИД': Идентификатор правила (например, ИД=1010) соответствует номеру операции в программном коде.
'Спр_А': Справочные данные из файла 'irregular_verbs.txt' - неправильные глаголы в трех формах: V1, V2 и V3.
'Спр_Б': Справочные данные из файла "only_ending_ed.txt" - любые слова, не только глаголы, с «неизменяемым окончанием -ed» .
'Спр_В': Справочные данные из файла 'verbs_ending_e.txt' - только глаголы, имеющие окончание -e. 
'Спр_C': Справочные данные из файла 'verbs_ending_non_ed.txt' - глаголы без окончания -ed, которые необходимо обработать раньше других правил, чтобы исключить случаи некорректной обработки. 

Подробную информацию по справочникам смотреть в файле: `3 Справочные данные приложения.txt`

Примечание:
При запуске приложения из командной строки с флагом -D:
    python myvocab.py -D
    myvocab.exe -D
отладочные логи, в том числе 'ИД' преобразований, записываются в файл app.log каталога `Myvocab_58b254sv`.

Чтобы начать обработку отдельных английских слов, в текстовый файл необходимо добавить строку с тегом <<word>>:


<<word>>

1) Неправильные глаголы в трех формах:

а) Глагол в форме V3 ('ИД'=1010) ('Спр_А'):
been    -> be


б) Глагол в форме V2 ('ИД'=1020) ('Спр_А')
was     -> be
were    -> be


в) Глагол в форме V1 ('ИД'=1030) ('Спр_А')
be 		-> be
feed	-> feed


2) Исключить из обработки слова ('ИД'=1040),
которые оканчиваются на -ed, в том числе и глаголы в базовой форме ('Спр_Б'):

bed 		-> bed
hundred 	-> hundred
creed 		-> creed
speed 		-> speed
unemployed 	-> unemployed


3) Если глагол оканчивается на -eed или -ed ('ИД'=1050),
то удалить -d, искать слово в справочнике ('Спр_В'):

braked 		-> brake
decreed 	-> decree
freed 		-> free
moved 		-> move

4) Если глагол оканчивается на -eed или -ed ('ИД'=1060),
то удалить -d, искать слово в справочнике ('Спр_C'):

willed 		-> will


5) Если глагол оканчивается на удвоенную согласную букву + -ed ('ИД'=1070),
то удалить -ed и убрать одну согласную:

focussed 	-> focus
planned 	-> plan
referred 	-> refer
travelled 	-> travel


6) Если глагол оканчивается на -ied ('ИД'=1080),
перед которым стоит согласная буква, то удалить -ed и заменить -i на -y:

copied 		-> copy
cried 		-> cry
married 	-> marry
tidied 		-> tidy


7) Если глагол оканчивается на -xed ('ИД'=1090),
то удалить -ed:

fixed	-> fix
mixed	-> mix


8) Если глагол оканчивается на -icked ('ИД'=1100),
то удалить -ked:

frolicked	-> frolic
picnicked 	-> picnic
trafficked 	-> traffic


9) Если глагол оканчивается на -ed ('ИД'=1110),
то удалить -ed:

allowed 	-> allow
labeled 	-> label
opened 		-> open
played 		-> play
walked 		-> walk
worshiped 	-> worship

10) Составные слова с окончанием -ed обрабатывается для каждого составного слова
по вышеприведенным правилам:

sun-dried       -> sun-dry
well-known      -> well-know
meat-fed        -> meat-feed
wind-powered    -> wind-power
middle-aged     -> middle-aged
short-handed    -> short-hand
strong-willed   -> strong-will
